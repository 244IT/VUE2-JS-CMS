# VUE2-JS-CMS



## 关于

Vue2Admin 是基于 Vue2、Vuex、VueRouter、Element 、JavaScript、Echart5 等后台系统解决方案。



是coderwhy老师VUE3+TS+CMS项目的Vue2重构版

网站参考地址：https://vue2jscms.chenhaohui.cn
